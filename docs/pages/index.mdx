# csp-toolkit

A comprehensive toolkit for working with Content Security Policy (CSP) directives in TypeScript.
We have a range of functions to help you work with CSP policies in your project.


## Features

- Typescript support for all CSP directives
- Nonce generation
- Hash generation

### Utility functions
- Policy merging
- Policy stringification
